Adds a Stephane video content type and Elementor gallery widgets.

== Description ==

Stephane Video Library adds:

* A `st_video` custom post type.
* Video categories and topics.
* Video meta fields and admin editing tools.
* A reusable frontend gallery renderer.
* Elementor widgets for gallery and featured video layouts.
* A shortcode for non-Elementor contexts.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin in WordPress admin.
3. If Elementor is active, use the "Stephane Widgets" category in Elementor.

== Shortcode ==

`[stvl_video_gallery limit="9" columns="3" show_search="yes"]`
